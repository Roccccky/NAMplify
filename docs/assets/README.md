# Landing-page screenshots

Save your app screenshots here with these exact names (the page `../index.html`
references them). PNG is fine. Keep them ≤ ~2000 px wide so the page loads fast.

| File name        | Which screenshot                                             |
|------------------|-------------------------------------------------------------|
| `hero-board.png` | Full clean pedalboard (no menu open) — the big hero image   |
| `tone3000.png`   | TONE3000 browser (list of captures / amps / IRs)            |
| `amp.png`        | Board with the AMP slot empty ("Choose capture…")           |
| `tuner.png`      | The tuner overlay (e.g. showing C2)                          |
| `settings.png`   | The Settings panel (audio, TONE3000, library)               |

After dropping the files in here:

```bash
cd "NAMplify Applikation"
git add docs/assets
git commit -m "Landing page: add app screenshots"
git push
```
